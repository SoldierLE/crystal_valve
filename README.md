# 项目说明

​	本项目以学习为目的，主要针对模拟电路设计进行学习。参考的书籍为铃木雅臣的《晶体管电路设计》。

​	本项目将会选取书本中每个章节较为重要的几个实用电路进行PCB制作及测试。
